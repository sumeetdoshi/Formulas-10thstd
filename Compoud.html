Simple Interest
  1. S.I = (P x R x T) / 100

  2. P = (S.I x 100) / R x T

  3. R = (S.I x 100) / P x T

  4. T = (S.I x 100) / R x P

Compound Intrest
  1. C.I = P (1+R/100)T - P

  2. A = P (1+R/100)T

    3. When Amount and principle are given


  3. C.I = A - P

  4. Discount = MP - SP
